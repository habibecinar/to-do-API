# To-Do API

A simple RESTful API built with Node.js and Express.js for managing to-do tasks. This API allows users to create, read, update, and delete tasks, providing a basic backend for task management applications.

## 🚀 Features

- **CRUD Operations**: Create, Read, Update, Delete tasks.
- **Data Persistence**: Store tasks in a MongoDB database.
- **Modular Routes**: Organized routing for scalability.
- **Environment Variables**: Configuration via `.env` file for sensitive data.
- **Error Handling**: Basic error handling for API requests.

## 🛠️ Technologies Used

- **Node.js**: JavaScript runtime for server-side development.
- **Express.js**: Web framework for building the API.
- **MongoDB**: NoSQL database for storing tasks.
- **Mongoose**: ODM for MongoDB to model the data.
- **dotenv**: Module to load environment variables from a `.env` file.
